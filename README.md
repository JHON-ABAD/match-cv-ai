# 🤖 MatchCV AI

[![Deploy to Vercel](https://img.shields.io/badge/Deployed_on-Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)]([AQUÍ_TU_LINK_DE_VERCEL])

> Optimización de perfiles IT impulsada por Inteligencia Artificial. MatchCV es una plataforma SaaS diseñada para analizar la compatibilidad de un currículum contra cualquier vacante del mercado, brindando feedback automático para superar los filtros ATS.

<img width="1919" height="908" alt="Captura de pantalla 2026-05-14 205505" src="https://github.com/user-attachments/assets/a338d829-876a-4982-96e9-5c11f195c621" />

## 🚀 Características Principales

* **Análisis con Inteligencia Artificial:** Integración directa con la API de Google Gemini para procesar lenguaje natural y evaluar perfiles.
* **Comparativa ATS:** Sistema de evaluación que simula los algoritmos de reclutamiento (Applicant Tracking Systems) para detectar palabras clave faltantes.
* **Feedback Accionable:** Generación de recomendaciones específicas en tiempo real para mejorar el CV según los requisitos del puesto.
* **Arquitectura Moderna:** Construido con renderizado del lado del servidor (SSR) para máxima velocidad y optimización.
